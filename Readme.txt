Installation
1. Clone or download the repository to your local machine.
2. Make sure you have Python installed on your system.
3. Install the required dependencies by running pip install tk tkcalendar.
Usage

1. Run the testfinal.py file to launch the application.
2. Fill in the required details for your task:
* Task name
* Start and end dates
* Assignees
* Project
* Description
* Priority
3. Click on the "Save draft" button to save the task as a draft or click on the "Submit" button to submit the task.
4. You can also attach files related to the task using the "Attach" button.
5. To clear all fields and start over, click on the "Cancel" button.
